Introduction to CSS Typography

Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed. CSS provides a wide range of properties to control the appearance of text on web pages.

In this project, we'll explore the various CSS properties that control text styling, including font families, sizes, weights, spacing, and more.

Font Family

The font-family property specifies the font for an element. You can specify multiple fonts as a "fallback" system.

font-family: 'Primary Font', 'Fallback Font', generic-family;

Common generic font families include:

  • serif - Fonts with small lines at the ends of characters
  • sans-serif - Fonts without lines at the ends of characters
  • monospace - Fonts where all characters have the same width
  • cursive - Fonts that mimic handwriting
  • fantasy - Decorative fonts

Font Size

The font-size property sets the size of the text. There are several units you can use:

/* Absolute units */ font-size: 16px; font-size: 12pt; /* Relative units */ font-size: 1em; /* Relative to parent element */ font-size: 1rem; /* Relative to root element */ font-size: 100%; /* Relative to parent element */ font-size: 2vw; /* Relative to viewport width */
