Introduction

Colors and backgrounds are essential for creating visually appealing web pages. CSS provides various ways to specify colors and apply backgrounds to elements.

In this tutorial, you'll learn:

  • Different formats for specifying colors in CSS
  • How to apply colors to text and backgrounds
  • How to work with background images
  • How to create gradient backgrounds
  • How to use multiple backgrounds

Color Formats

CSS supports several formats for specifying colors:

1. Color Names

CSS provides 140 named colors like red, blue, green, etc.

Color Name Examples:

red
blue
green
yellow
purple
.element-1 { color: red; }
.element-2 { background-color: blue; }

2. Hexadecimal Colors

Hexadecimal color codes start with a hash (#) followed by six hexadecimal digits representing RGB values.

Hexadecimal Color Examples:

#FF0000
Red
#00FF00
Green
#0000FF
Blue
#9C27B0
Purple
#FF9800
Orange
.element-1 { color: #FF0000; } /* Red */
.element-2 { background-color: #00FF00; } /* Green */

Shorthand hex: If all six digits are made up of three repeated pairs, you can use a shorthand with just three digits.

#FF0000 can be written as #F00
#00FF00 can be written as #0F0
#0000FF can be written as #00F

3. RGB Colors

RGB colors are specified using the rgb() function with values for red, green, and blue (0-255).

RGB Color Examples:

rgb(255, 0, 0)
Red
rgb(0, 255, 0)
Green
rgb(0, 0, 255)
Blue
rgb(156, 39, 176)
Purple
rgb(255, 152, 0)
Orange
.element-1 { color: rgb(255, 0, 0); } /* Red */
.element-2 { background-color: rgb(0, 255, 0); } /* Green */

4. RGBA Colors

RGBA colors are similar to RGB but include an alpha channel for transparency (0-1).

RGBA Color Examples (Red with different opacity levels):

rgba(255, 0, 0, 1)
100%
rgba(255, 0, 0, 0.8)
80%
rgba(255, 0, 0, 0.6)
60%
rgba(255, 0, 0, 0.4)
40%
rgba(255, 0, 0, 0.2)
20%
.element-1 { color: rgba(255, 0, 0, 1); } /* Fully opaque red */
.element-2 { background-color: rgba(0, 0, 255, 0.5); } /* Semi-transparent blue */

5. HSL Colors

HSL colors are specified using the hsl() function with values for hue (0-360), saturation (0-100%), and lightness (0-100%).

HSL Color Examples:

hsl(0, 100%, 50%)
Red
hsl(120, 100%, 50%)
Green
hsl(240, 100%, 50%)
Blue
hsl(300, 100%, 50%)
Magenta
hsl(39, 100%, 50%)
Orange
.element-1 { color: hsl(0, 100%, 50%); } /* Red */
.element-2 { background-color: hsl(120, 100%, 50%); } /* Green */

HSL also has an HSLA variant that includes an alpha channel for transparency.

.element { background-color: hsla(240, 100%, 50%, 0.5); } /* Semi-transparent blue */

Background Colors

You can set the background color of elements using the background-color property.

Background Color Example:

This element has a light purple background color.

.bg-color {
    background-color: #e1bee7;
    padding: 20px;
}

Background Images

You can add background images to elements using the background-image property.

Basic Background Image:

.bg-image {
    background-image: url('image.jpg');
    height: 200px;
}

Background Repeat:

By default, background images repeat both horizontally and vertically. You can control this with the background-repeat property.

Default (repeat):
repeat-x:
repeat-y:
no-repeat:
.bg-repeat { background-image: url('image.jpg'); } /* Default: repeat */
.bg-repeat-x { background-image: url('image.jpg'); background-repeat: repeat-x; }
.bg-repeat-y { background-image: url('image.jpg'); background-repeat: repeat-y; }
.bg-no-repeat { background-image: url('image.jpg'); background-repeat: no-repeat; }

Background Position:

You can control the position of a background image using the background-position property.

.bg-position {
    background-image: url('image.jpg');
    background-repeat: no-repeat;
    background-position: center; /* Can also use: top, bottom, left, right, or specific values like "20px 30px" */
}

Background Size:

You can control the size of a background image using the background-size property.

cover:
contain:
.bg-size-cover {
    background-image: url('image.jpg');
    background-size: cover; /* Scales the image to cover the entire element */
    background-position: center;
}

.bg-size-contain {
    background-image: url('image.jpg');
    background-size: contain; /* Scales the image to fit inside the element */
    background-repeat: no-repeat;
    background-position: center;
}

Background Attachment:

You can control how a background image scrolls with the content using the background-attachment property.

.bg-attachment-fixed {
    background-image: url('image.jpg');
    background-attachment: fixed; /* Image stays fixed when scrolling */
    background-size: cover;
}

Gradients

CSS gradients let you display smooth transitions between two or more colors.

Linear Gradients:

Linear gradients transition colors along a straight line.

.gradient-linear-1 {
    background: linear-gradient(to right, red, yellow);
}

.gradient-linear-2 {
    background: linear-gradient(45deg, blue, purple);
}

.gradient-linear-3 {
    background: linear-gradient(to bottom, #9c27b0, #e1bee7);
}

Radial Gradients:

Radial gradients transition colors from a central point outward.

.gradient-radial {
    background: radial-gradient(circle, yellow, red);
}

Multiple Color Stops:

Gradients can have multiple color stops.

.gradient-multi {
    background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
}

Multiple Backgrounds

CSS allows you to apply multiple backgrounds to a single element. They are stacked with the first background on top.

.multiple-bg {
    background: 
        url('image1.jpg') no-repeat top left,
        url('image2.jpg') no-repeat top right,
        url('image3.jpg') no-repeat bottom left,
        url('image4.jpg') no-repeat bottom right,
        linear-gradient(to right, rgba(156, 39, 176, 0.3), rgba(156, 39, 176, 0.3));
}
