Introduction to CSS Preprocessors

CSS preprocessors are scripting languages that extend the default capabilities of CSS. They enable features like variables, nesting, mixins, inheritance, and more, which make CSS code more maintainable and easier to write.

Popular CSS preprocessors include:

  • Sass/SCSS - The most mature and widely used preprocessor
  • Less - A JavaScript-based preprocessor
  • Stylus - A flexible preprocessor with a minimalist syntax

In this project, we'll focus primarily on Sass/SCSS, which is the most widely adopted preprocessor.

Variables in Preprocessors

Variables allow you to store values that you want to reuse throughout your stylesheet. This makes it easier to maintain consistent colors, spacing, and other values.

CSS
/* CSS Variables (Custom Properties) */ :root { --primary-color: #9c27b0; --secondary-color: #2196f3; --spacing: 20px; } .button { background-color: var(--primary-color); padding: var(--spacing); } .header { color: var(--primary-color); margin-bottom: var(--spacing); }
SCSS
/* SCSS Variables */ $primary-color: #9c27b0; $secondary-color: #2196f3; $spacing: 20px; .button { background-color: $primary-color; padding: $spacing; } .header { color: $primary-color; margin-bottom: $spacing; }

Nesting in Preprocessors

Nesting allows you to write CSS selectors that follow the same visual hierarchy as your HTML. This makes your code more readable and reduces repetition.

CSS
/* CSS */ .nav { background-color: #333; } .nav ul { list-style: none; margin: 0; padding: 0; } .nav li { display: inline-block; } .nav a { color: white; text-decoration: none; } .nav a:hover { color: #9c27b0; }
SCSS
/* SCSS with Nesting */ .nav { background-color: #333; ul { list-style: none; margin: 0; padding: 0; } li { display: inline-block; } a { color: white; text-decoration: none; &:hover { color: #9c27b0; } } }

Mixins in Preprocessors

Mixins are reusable blocks of CSS declarations that can be included in other rules. They can also accept parameters, making them even more flexible.

CSS
/* CSS */ .button-primary { background-color: #9c27b0; color: white; padding: 10px 20px; border-radius: 4px; transition: background-color 0.3s; } .button-secondary { background-color: #2196f3; color: white; padding: 10px 20px; border-radius: 4px; transition: background-color 0.3s; }
SCSS
/* SCSS with Mixins */ @mixin button($bg-color) { background-color: $bg-color; color: white; padding: 10px 20px; border-radius: 4px; transition: background-color 0.3s; } .button-primary { @include button(#9c27b0); } .button-secondary { @include button(#2196f3); }
