CSS Pseudo-classes

Use pseudo-classes to style elements based on their state or position

Introduction to CSS Pseudo-classes

Pseudo-classes are special selectors that target elements based on their state, position in the document, or other characteristics that cannot be targeted with simple selectors.

They allow you to style elements dynamically based on user interaction, form states, structural position, and more.

Link Pseudo-classes

Link pseudo-classes target links in different states.

a:link { color: blue; } /* Unvisited links */ a:visited { color: purple; } /* Visited links */ a:hover { color: red; } /* Mouse over links */ a:active { color: green; } /* Active links (being clicked) */

User Action Pseudo-classes

These pseudo-classes respond to user interactions with elements.

element:hover { ... } /* When mouse is over the element */ element:active { ... } /* When element is being activated */ element:focus { ... } /* When element has keyboard focus */

Form Pseudo-classes

These pseudo-classes target form elements based on their state or attributes.

input:required { ... } /* Required form elements */ input:optional { ... } /* Optional form elements */ input:valid { ... } /* Form elements with valid input */ input:invalid { ... } /* Form elements with invalid input */ input:checked { ... } /* Checked radio buttons or checkboxes */
