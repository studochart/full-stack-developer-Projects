CSS Media Queries

Create responsive designs with media queries for different devices

Introduction to Media Queries

Media queries allow you to apply different CSS styles based on various device characteristics such as screen width, height, orientation, and more. They are a fundamental component of responsive web design.

With media queries, you can create websites that adapt to different screen sizes and provide an optimal viewing experience across a wide range of devices, from desktop computers to mobile phones.

Basic Media Query Syntax

The basic syntax for a media query includes a media type and one or more expressions that check for specific conditions:

@media [media-type] and ([media-feature]) { /* CSS rules to apply when conditions are met */ } /* Examples */ @media screen and (max-width: 768px) { /* Styles for screens 768px wide or less */ } @media screen and (min-width: 769px) and (max-width: 1024px) { /* Styles for screens between 769px and 1024px */ } @media (orientation: landscape) { /* Styles for landscape orientation */ } @media print { /* Styles for print */ }
