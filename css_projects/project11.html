Introduction to CSS Effects

CSS offers a variety of properties to create visual effects that enhance the appearance of web elements. These effects include shadows, gradients, transforms, filters, and blend modes.

In this project, we'll explore these different effects and how they can be combined to create visually appealing designs.

Shadow Effects

CSS provides two main shadow properties: box-shadow for elements and text-shadow for text.

/* Box Shadow Syntax */ box-shadow: [horizontal offset] [vertical offset] [blur radius] [spread radius] [color]; /* Text Shadow Syntax */ text-shadow: [horizontal offset] [vertical offset] [blur radius] [color];

Box Shadow Examples

Basic Shadow

box-shadow: 0 4px 8px rgba(0,0,0,0.1);

Large Shadow

box-shadow: 0 10px 30px rgba(0,0,0,0.2);

Inset Shadow

box-shadow: inset 0 0 10px rgba(0,0,0,0.2);

Colored Shadow

box-shadow: 0 5px 15px rgba(156,39,176,0.4);

Text Shadow Examples

Basic Text Shadow

text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

Glowing Text

text-shadow: 0 0 10px rgba(156,39,176,0.8);

Outline Text

text-shadow: -1px -1px 0 #9c27b0, 1px -1px 0 #9c27b0, -1px 1px 0 #9c27b0, 1px 1px 0 #9c27b0;
