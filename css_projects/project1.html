Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS is one of the core languages of the web and is standardized by the W3C (World Wide Web Consortium).

Why CSS is Important:

  • Separates content (HTML) from presentation (CSS)
  • Allows consistent styling across multiple pages
  • Enables responsive design for different devices
  • Improves accessibility and user experience
  • Reduces code duplication and maintenance

CSS Syntax

CSS consists of a selector and a declaration block:

selector {
    property: value;
    property: value;
    /* More property-value pairs */
}
  • Selector: Points to the HTML element you want to style
  • Declaration Block: Contains one or more declarations separated by semicolons
  • Property: The style attribute you want to change
  • Value: The value for the property

Example:

p {
    color: blue;
    font-size: 16px;
    margin: 10px;
}

This CSS rule sets the text color to blue, font size to 16 pixels, and margin to 10 pixels for all paragraph elements.

CSS Selectors

Selectors are patterns used to select the HTML elements you want to style. Here are the most common types:

Selector Example Description
Element Selector p { } Selects all <p> elements
Class Selector .intro { } Selects all elements with class="intro"
ID Selector #header { } Selects the element with id="header"
Universal Selector * { } Selects all elements
Grouping Selector h1, h2, p { } Selects all h1, h2, and p elements
Descendant Selector div p { } Selects all <p> elements inside <div> elements
Child Selector div > p { } Selects all <p> elements that are direct children of a <div> element

Selector Examples:

This div has an ID of "id-example"

This is a paragraph element

This is a div containing a span element as a direct child

CSS Properties

CSS has hundreds of properties that control different aspects of an element's appearance. Here are some common categories:

Text Properties

  • color: Sets the color of text
  • font-family: Specifies the font
  • font-size: Sets the size of the font
  • font-weight: Sets how thick or thin characters are displayed
  • text-align: Sets the horizontal alignment of text
  • text-decoration: Adds decoration to text (underline, overline, etc.)

Box Properties

  • width, height: Sets the width and height of an element
  • margin: Sets the margin area on all four sides of an element
  • padding: Sets the padding area on all four sides of an element
  • border: Sets the border on all four sides of an element
  • background-color: Sets the background color of an element

CSS Specificity

Specificity determines which CSS rule is applied when multiple rules target the same element. The more specific selector wins.

Specificity Hierarchy (from lowest to highest):

  1. Element selectors (p, div, etc.)
  2. Class selectors (.example), attribute selectors ([type="text"]), and pseudo-classes (:hover)
  3. ID selectors (#example)
  4. Inline styles (style="color: red;")
  5. !important declaration (overrides all other styles)

Specificity Example:

This paragraph has multiple styles targeting it.

CSS Used:

#specificity-example p {
    color: black; /* Less specific */
}

#specificity-example .text {
    color: blue; /* More specific than element selector */
}

#specificity-example #unique {
    color: red; /* Most specific */
}

The text is red because the ID selector has the highest specificity among these three selectors.

CSS Inheritance

Some CSS properties are inherited from parent elements to their children, while others are not.

Properties that are typically inherited:

  • Text properties (color, font-family, font-size, etc.)
  • Line height
  • List styles
  • Text alignment

Properties that are typically not inherited:

  • Width, height
  • Margin, padding
  • Border
  • Background
  • Position

Inheritance Example:

This paragraph inherits the purple color and Georgia font from its parent.

This paragraph also inherits the text styles, but the border, margin, and padding on its parent div are not inherited.

CSS Used:

    color: purple;
    font-family: Georgia, serif;
}

    border: 2px solid purple;
    margin: 10px;
    padding: 10px;
}

Adding CSS to HTML

There are three ways to add CSS to an HTML document:

1. Inline CSS

Using the style attribute directly on HTML elements:

<p style="color: blue; font-size: 16px;">This is a paragraph with inline styles.</p>

2. Internal CSS

Using a <style> element in the <head> section:

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

3. External CSS

Using a separate CSS file linked with the <link> element:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

And in the styles.css file:

p {
    color: blue;
    font-size: 16px;
}

Best Practice:

External CSS is generally the best approach because it:

  • Separates HTML and CSS
  • Makes maintenance easier
  • Allows the same CSS file to be used across multiple HTML pages
  • Takes advantage of browser caching

Practice Exercise

Try creating a simple HTML file with the following elements and style them using CSS:

  1. A heading with a custom color and font
  2. A paragraph with a different color and font size
  3. A div with a border, background color, and padding
  4. A list with custom bullet points or numbers
  5. A link that changes color when hovered over

You can use any of the three methods to add CSS (inline, internal, or external), but external CSS is recommended for practice.
