Introduction to CSS Grid

CSS Grid Layout is a two-dimensional layout system designed for the web. It lets you lay out items in rows and columns, and has many features that make building complex layouts straightforward.

Grid is the most powerful layout system available in CSS, allowing for precise control over both rows and columns simultaneously.

Basic Grid Concepts

To create a grid container, you set the display property to grid or inline-grid. All direct children of the grid container become grid items.

.container { display: grid; grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */ grid-gap: 20px; /* Gap between grid items */ }

Grid Template Columns

The grid-template-columns property defines the number and width of columns in your grid.

/* Three columns with different widths */ grid-template-columns: 1fr 2fr 1fr; /* Using repeat() function */ grid-template-columns: repeat(3, 1fr); /* Mixed units */ grid-template-columns: 200px 1fr 2fr;
