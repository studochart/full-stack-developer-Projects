CSS Variables

Use CSS custom properties (variables) for more maintainable stylesheets

Introduction to CSS Variables

CSS variables, also known as custom properties, allow you to store specific values to be reused throughout your stylesheet. They help make your CSS more maintainable, flexible, and consistent.

With CSS variables, you can define values once and use them multiple times, making it easier to update your styles globally by changing just one value.

Defining and Using CSS Variables

CSS variables are defined using the --variable-name syntax and are accessed using the var(--variable-name) function:

/* Defining variables in the :root selector (global scope) */ :root { --primary-color: #9c27b0; --spacing-medium: 20px; --font-size-large: 1.5rem; } /* Using variables */ .element { color: var(--primary-color); padding: var(--spacing-medium); font-size: var(--font-size-large); } /* Using variables with fallback values */ .another-element { margin: var(--spacing-small, 10px); }
