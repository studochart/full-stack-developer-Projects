HTML Quotations and Citations

HTML provides several elements for marking up quotations, citations, and other referenced content. These elements add semantic meaning to your content and can be styled differently from regular text.

Blockquote Element

The <blockquote> element is used for extended quotations that are typically displayed as indented blocks of text. It's ideal for longer quotes that stand apart from the main text.

Basic Blockquote

The greatest glory in living lies not in never falling, but in rising every time we fall.

HTML code:

<blockquote>
    <p>The greatest glory in living lies not in never falling, but in rising every time we fall.</p>
</blockquote>

Blockquote with Citation

The greatest glory in living lies not in never falling, but in rising every time we fall.

— Nelson Mandela

HTML code:

<blockquote cite="https://www.brainyquote.com/quotes/nelson_mandela_378967">
    <p>The greatest glory in living lies not in never falling, but in rising every time we fall.</p>
    <footer>— <cite>Nelson Mandela</cite></footer>
</blockquote>

Q Element (Inline Quotes)

The <q> element is used for shorter, inline quotations. Browsers typically add quotation marks around the content of a <q> element.

Basic Q Element

According to Einstein, Imagination is more important than knowledge.

HTML code:

<p>According to Einstein, <q>Imagination is more important than knowledge.</q></p>

Q Element with Citation

Shakespeare wrote, To be or not to be, that is the question.

HTML code:

<p>Shakespeare wrote, <q cite="http://www.shakespeare.org.uk">To be or not to be, that is the question.</q></p>

Cite Element

The <cite> element is used to reference a creative work, such as a book, movie, play, poem, song, or website. The content of the <cite> element is typically rendered in italics.

Basic Cite Element

My favorite book is The Lord of the Rings by J.R.R. Tolkien.

HTML code:

<p>My favorite book is <cite>The Lord of the Rings</cite> by J.R.R. Tolkien.</p>

Cite with Other Elements

The movie The Shawshank Redemption contains the famous quote: Get busy living, or get busy dying.

HTML code:

<p>The movie <cite>The Shawshank Redemption</cite> contains the famous quote: <q>Get busy living, or get busy dying.</q></p>

Cite Attribute

The cite attribute (not to be confused with the <cite> element) can be used with <blockquote> and <q> elements to specify the source URL of a quotation.

The cite attribute is not visible to the user, but it provides a way to link to the source of the quotation. It's useful for search engines and other applications that can make use of this metadata.

Example with Blockquote

Be yourself; everyone else is already taken.

— Oscar Wilde

HTML code:

<blockquote cite="https://www.goodreads.com/quotes/1173-be-yourself-everyone-else-is-already-taken">
    <p>Be yourself; everyone else is already taken.</p>
    <footer>— <cite>Oscar Wilde</cite></footer>
</blockquote>

Abbreviations

The <abbr> element is used to mark up abbreviations and acronyms. It can include a title attribute to provide the full expansion of the term.

Basic Abbreviation

The WHO was founded in 1948.

HTML code:

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

Abbreviations in Text

I need to finish this project ASAP. The deadline is EOD Friday.

HTML code:

<p>I need to finish this project <abbr title="As Soon As Possible">ASAP</abbr>. The deadline is <abbr title="End Of Day">EOD</abbr> Friday.</p>

Address Element

The <address> element is used to provide contact information for the author or owner of a document or article. It's typically rendered in italics.

Basic Address

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

HTML code:

<address>
    Written by John Doe.<br>
    Visit us at:<br>
    Example.com<br>
    Box 564, Disneyland<br>
    USA
</address>

Address with Links

Article by John Doe.
Follow me on Twitter.

HTML code:

<address>
    Article by <a href="mailto:john@example.com">John Doe</a>.<br>
    Follow me on <a href="https://twitter.com/johndoe">Twitter</a>.
</address>

BDO Element (Bi-Directional Override)

The <bdo> element is used to override the current text direction. This is useful when including text in languages that are written right-to-left, such as Arabic or Hebrew.

Left-to-Right Text

This text will go from left to right.

HTML code:

<p><bdo dir="ltr">This text will go from left to right.</bdo></p>

Right-to-Left Text

This text will go from right to left.

HTML code:

<p><bdo dir="rtl">This text will go from right to left.</bdo></p>

Practical Examples

Example 1: Academic Citation

The Impact of Climate Change

Climate change is one of the most pressing issues of our time. According to recent research:

It is unequivocal that human influence has warmed the atmosphere, ocean and land. Widespread and rapid changes in the atmosphere, ocean, cryosphere and biosphere have occurred.

— IPCC Sixth Assessment Report, 2021

This finding underscores the urgency of taking action to reduce greenhouse gas emissions.

Example 2: Literary Analysis

Symbolism in The Great Gatsby

F. Scott Fitzgerald's The Great Gatsby is rich with symbolism. One of the most famous passages describes the green light at the end of Daisy's dock:

Gatsby believed in the green light, the orgastic future that year by year recedes before us. It eluded us then, but that's no matter—tomorrow we will run faster, stretch out our arms farther...

— F. Scott Fitzgerald, The Great Gatsby

The green light symbolizes Gatsby's hopes and dreams, particularly his hope for a future with Daisy.
