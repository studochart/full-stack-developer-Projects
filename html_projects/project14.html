Basic Table Structure

A basic HTML table consists of the table element with tr (table row) elements, which contain td (table data) elements:

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

HTML code for this table:

<table>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>

Table Headers

Table headers are defined with the th element instead of td. Headers are typically bold and centered by default:

Name Age Country
John Doe 28 USA
Jane Smith 32 Canada

HTML code for this table:

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>28</td>
        <td>USA</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>32</td>
        <td>Canada</td>
    </tr>
</table>

Column and Row Spanning

You can make a cell span multiple columns with the colspan attribute or multiple rows with the rowspan attribute:

Best Practices for Tables

  • Use tables for tabular data only, not for layout
  • Include proper table headers with th elements
  • Use caption to provide a title for the table
  • Use thead, tbody, and tfoot to organize table sections
  • Keep tables simple and avoid excessive nesting
  • Use CSS for styling rather than HTML attributes
  • Consider accessibility by using proper markup and providing context
