HTML Tables - Basic Structure

In this lesson, you'll learn how to create tables in HTML using the <table>, <tr>, <th>, and <td> elements.

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>28</td>
        <td>USA</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>32</td>
        <td>Canada</td>
    </tr>
</table>
