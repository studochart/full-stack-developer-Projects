HTML Attributes

HTML attributes provide additional information about HTML elements. They are always specified in the start tag and usually come in name/value pairs like: name="value". This page demonstrates various HTML attributes and how they modify elements.

Attribute Basics

HTML attributes have the following characteristics:

  • Attributes provide additional information about an element
  • They are always specified in the start tag
  • They usually come in name/value pairs like: name="value"
  • Attribute values should always be enclosed in quotes (single or double)
  • Some attributes can be minimized (boolean attributes)

Basic syntax:

<element attribute="value">Content</element>

Example:

<a href="https://www.example.com" target="_blank">Visit Example.com</a>

In this example, href and target are attributes of the <a> element.

Global Attributes

Global attributes are attributes that can be used with any HTML element. Here are some of the most common ones:

id

The id attribute specifies a unique id for an element. The value must be unique within the HTML document.

<div id="header">This is the header</div>
This element has an id of "header-example"

class

The class attribute specifies one or more class names for an element. Multiple classes can be specified by separating them with spaces.

<p class="important highlight">This is an important paragraph.</p>

This paragraph has classes "important" and "highlight"

style

The style attribute specifies inline CSS styles for an element.

<p style="color: blue; font-size: 18px;">This is a styled paragraph.</p>

This is a styled paragraph.

title

The title attribute specifies extra information about an element, shown as a tooltip when the mouse hovers over the element.

<abbr title="HyperText Markup Language">HTML</abbr>

The HTML language is used to create web pages. (Hover over "HTML" to see the title)

lang

The lang attribute specifies the language of the element's content.

<p lang="fr">Bonjour le monde</p>

Bonjour le monde

dir

The dir attribute specifies the text direction of the element's content.

<p dir="rtl">This text will be displayed right-to-left.</p>

This text will be displayed right-to-left.

hidden

The hidden attribute specifies that an element is not yet, or is no longer, relevant.

<p hidden>This paragraph is hidden.</p>

This paragraph is hidden.

tabindex

The tabindex attribute specifies the tab order of an element.

<input type="text" tabindex="1">
<input type="text" tabindex="2">

Element-Specific Attributes

Many HTML elements have specific attributes that only apply to them. Here are some examples:

a element

  • href - Specifies the URL of the page the link goes to
  • target - Specifies where to open the linked document
  • rel - Specifies the relationship between the current document and the linked document
<a href="https://www.example.com" target="_blank" rel="noopener">Visit Example.com</a>

Visit Example.com

img element

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image
  • width and height - Specify the width and height of the image
<img src="image.jpg" alt="Description of image" width="300" height="200">

input element

  • type - Specifies the type of input
  • name - Specifies the name of the input
  • value - Specifies the initial value of the input
  • placeholder - Specifies a hint that describes the expected value
  • required - Specifies that the input field must be filled out
<input type="text" name="username" placeholder="Enter your username" required>

button element

  • type - Specifies the type of button
  • disabled - Specifies that the button should be disabled
<button type="button">Click Me</button>
<button type="button" disabled>Disabled Button</button>

Boolean Attributes

Boolean attributes are attributes that don't require a value. The presence of the attribute implies a "true" value, and the absence implies a "false" value.

Examples of boolean attributes:

  • disabled - Disables an input element
  • checked - Pre-selects a checkbox or radio button
  • readonly - Makes an input field read-only
  • required - Makes an input field required
  • hidden - Hides an element
  • selected - Pre-selects an option in a dropdown

Boolean attributes can be written in several ways:

<input type="checkbox" checked>
<input type="checkbox" checked="checked">
<input type="checkbox" checked="">

All three examples above are equivalent and represent a checked checkbox:

Checked checkbox

More examples:

<input type="text" readonly>
<button disabled>Disabled Button</button>
<select>
    <option selected>Selected Option</option>
    <option>Another Option</option>
</select>

Event Attributes

Event attributes allow you to run JavaScript when certain events occur. They start with "on" followed by the event name.

Common event attributes:

  • onclick - Triggered when the element is clicked
  • onmouseover - Triggered when the mouse pointer moves over the element
  • onmouseout - Triggered when the mouse pointer moves out of the element
  • onkeydown - Triggered when a key is pressed down
  • onload - Triggered when the element has finished loading
  • onsubmit - Triggered when a form is submitted

Examples:

<button onclick="alert('Button clicked!')">Click Me</button>
<div onmouseover="this.style.backgroundColor='yellow'" onmouseout="this.style.backgroundColor='white'">
    Hover over me
</div>

Custom Data Attributes

HTML5 allows you to create custom attributes using the data-* prefix. These attributes can store extra information that doesn't have a visual representation.

Custom data attributes:

  • Must start with data-
  • Can contain only lowercase letters, numbers, hyphens, dots, colons, and underscores
  • Can be accessed via JavaScript using the dataset property

Example:

<div id="user" data-id="123" data-user="john" data-role="admin">John Doe</div>
John Doe (This div has custom data attributes: data-id="123", data-user="john", data-role="admin")

Accessing custom data attributes with JavaScript:

// HTML: <div id="user" data-id="123" data-user="john" data-role="admin">John Doe</div>

// JavaScript:
const user = document.getElementById('user');
console.log(user.dataset.id); // "123"
console.log(user.dataset.user); // "john"
console.log(user.dataset.role); // "admin"

Practical Examples

Example 1: Link with Multiple Attributes

<a href="https://www.example.com" 
   target="_blank" 
   rel="noopener noreferrer" 
   title="Visit Example.com in a new tab"
   class="external-link"
   id="example-link">
   Visit Example.com
</a>

Visit Example.com

Example 2: Form with Various Input Attributes

<form action="/submit" method="post" id="contact-form">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" placeholder="Enter your name" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="Enter your email" required>
    
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" cols="50" placeholder="Enter your message"></textarea>
    
    <input type="checkbox" id="subscribe" name="subscribe" checked>
    <label for="subscribe">Subscribe to newsletter</label>
    
    <button type="submit">Submit</button>
</form>
