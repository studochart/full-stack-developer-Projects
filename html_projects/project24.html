HTML Colors

Colors play a crucial role in web design. HTML allows you to specify colors in various ways to style elements like text, backgrounds, borders, and more. This page demonstrates different color formats and how to apply them in HTML.

Color Formats

HTML and CSS support several different formats for specifying colors:

  • Color Names: Predefined color names like "red", "blue", "green", etc.
  • Hexadecimal: Six-digit codes preceded by a hash (#) like #FF0000 for red
  • RGB: Red, Green, Blue values specified using rgb(r, g, b)
  • RGBA: RGB with an Alpha (transparency) channel using rgba(r, g, b, a)
  • HSL: Hue, Saturation, Lightness values using hsl(h, s%, l%)
  • HSLA: HSL with an Alpha channel using hsla(h, s%, l%, a)

Hexadecimal Colors

Hexadecimal color codes start with a hash (#) followed by six hexadecimal digits. The first two digits represent red, the middle two represent green, and the last two represent blue.

Color Hex Code Example
Red #FF0000
Green #00FF00
Blue #0000FF
Yellow #FFFF00
Cyan #00FFFF
Magenta #FF00FF
Black #000000
White #FFFFFF

Shorthand Hex

If all six digits are made up of three repeated pairs, you can use a shorthand notation with just three digits:

Full Hex Shorthand Example
#FF0000 #F00
#00FF00 #0F0
#0000FF #00F
#FFFF00 #FF0

Example usage:

<p style="color: #FF0000;">This text is red using a full hex code.</p>
<p style="color: #F00;">This text is also red using shorthand hex.</p>

RGB Colors

RGB colors are specified using the rgb() function, which takes three parameters for red, green, and blue. Each parameter can range from 0 to 255.

Color RGB Value Example
Red rgb(255, 0, 0)
Green rgb(0, 255, 0)
Blue rgb(0, 0, 255)
Yellow rgb(255, 255, 0)
Purple rgb(128, 0, 128)
Teal rgb(0, 128, 128)
Gray rgb(128, 128, 128)

Example usage:

<p style="color: rgb(0, 0, 255);">This text is blue using RGB.</p>
<div style="background-color: rgb(255, 165, 0);">This div has an orange background using RGB.</div>

This text is blue using RGB.

RGBA Colors (with Transparency)

RGBA colors are similar to RGB but include an alpha channel for transparency. The alpha value ranges from 0 (completely transparent) to 1 (completely opaque).

Color RGBA Value Example
Red (50% transparent) rgba(255, 0, 0, 0.5)
Green (70% transparent) rgba(0, 255, 0, 0.3)
Blue (20% transparent) rgba(0, 0, 255, 0.8)
Black (90% transparent) rgba(0, 0, 0, 0.1)

Example usage:

<div style="background-color: rgba(255, 0, 0, 0.5); padding: 10px;">
    This div has a semi-transparent red background.
</div>

HSL Colors

HSL stands for Hue, Saturation, and Lightness. It's an alternative way to specify colors that can be more intuitive for some design tasks.

  • Hue is a degree on the color wheel (0 to 360), where 0 is red, 120 is green, and 240 is blue.
  • Saturation is a percentage value, where 0% is a shade of gray and 100% is the full color.
  • Lightness is also a percentage, where 0% is black and 100% is white. 50% is the "normal" color.
Color HSL Value Example
Red hsl(0, 100%, 50%)
Green hsl(120, 100%, 50%)
Blue hsl(240, 100%, 50%)
Yellow hsl(60, 100%, 50%)
Purple hsl(270, 100%, 50%)
Light Blue hsl(195, 100%, 50%)
Pink hsl(330, 100%, 80%)

Example usage:

<p style="color: hsl(240, 100%, 50%);">This text is blue using HSL.</p>

This text is blue using HSL.

Applying Colors in HTML

There are several ways to apply colors in HTML:

1. Using the style Attribute

You can apply colors directly to HTML elements using the style attribute:

<p style="color: blue;">Blue text</p>
<p style="background-color: yellow;">Text with yellow background</p>
<p style="color: white; background-color: black;">White text on black background</p>

Blue text

Text with yellow background

White text on black background

2. Using HTML Color Attributes (Deprecated)

Some HTML elements used to support color attributes directly, but these are now deprecated in favor of CSS:

<font color="red">Red text</font> <!-- Deprecated -->
<body bgcolor="lightblue"> <!-- Deprecated -->

Instead of using deprecated attributes, use CSS for all color styling.

3. Using CSS (Recommended)

The best practice is to use CSS for styling, either in a separate stylesheet or in a style tag in the head section:

<style>
    p {
        color: blue;
    }
    .highlight {
        background-color: yellow;
    }
    #special {
        color: white;
        background-color: black;
    }
</style>

<p>Blue text</p>
<p class="highlight">Text with yellow background</p>
<p id="special">White text on black background</p>

Color Schemes and Accessibility

Color Contrast

When choosing colors, it's important to ensure sufficient contrast between text and background for readability. The Web Content Accessibility Guidelines (WCAG) recommend a contrast ratio of at least 4.5:1 for normal text and 3:1 for large text.

Good contrast: White text on black background

Good contrast: Black text on yellow background

Good contrast: Yellow text on red background

Poor contrast: Green text on red background

Color Blindness Considerations

About 8% of men and 0.5% of women have some form of color blindness. When designing with color:

  • Don't rely solely on color to convey information
  • Use patterns or labels in addition to colors
  • Avoid problematic color combinations (red/green, blue/purple, etc.)
