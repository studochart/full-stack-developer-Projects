Image Formats

Common web image formats include:

  • JPEG/JPG - Best for photographs and complex images with many colors
  • PNG - Best for images that need transparency or have text/sharp edges
  • GIF - Supports animation, limited to 256 colors
  • SVG - Vector format, scalable without quality loss, ideal for logos and icons
  • WebP - Modern format with better compression and quality than JPEG and PNG

Best Practices for Images

  • Always include the alt attribute for accessibility
  • Specify width and height to prevent layout shifts
  • Optimize images for web to reduce file size
  • Use appropriate image formats for different types of images
  • Make images responsive for different screen sizes
  • Consider lazy loading for images below the fold
