HTML Lists

HTML provides three types of lists to organize and structure content:

  • Unordered Lists (ul)
  • Ordered Lists (ol)
  • Description Lists (dl)

Unordered Lists

Unordered lists are used when the order of items doesn't matter. Each item starts with a bullet point by default.

Basic Unordered List

Here's a simple shopping list:

  • Milk
  • Bread
  • Eggs
  • Cheese
  • Vegetables

Nested Unordered Lists

You can nest lists inside other lists:

  • Fruits
    • Apples
    • Bananas
    • Oranges
  • Vegetables
    • Carrots
    • Broccoli
    • Spinach
  • Dairy
    • Milk
    • Cheese
    • Yogurt

Ordered Lists

Ordered lists are used when the sequence of items matters. Each item is numbered by default.

Basic Ordered List

Steps to make a sandwich:

  1. Get two slices of bread
  2. Spread butter or mayonnaise on one side of each slice
  3. Add your favorite fillings
  4. Put the slices together
  5. Cut the sandwich in half (optional)

Ordered List with Different Numbering

You can change the numbering style using the type attribute:

Type "A" - Uppercase Letters

  1. First item
  2. Second item
  3. Third item

Type "a" - Lowercase Letters

  1. First item
  2. Second item
  3. Third item

Type "I" - Uppercase Roman Numerals

  1. First item
  2. Second item
  3. Third item

Type "i" - Lowercase Roman Numerals

  1. First item
  2. Second item
  3. Third item

Starting with a Different Number

You can start the list with a different number using the start attribute:

  1. This is item #5
  2. This is item #6
  3. This is item #7

Nested Ordered Lists

You can also nest ordered lists:

  1. Main Chapter
    1. Section 1
    2. Section 2
    3. Section 3
  2. Second Chapter
    1. Section 1
    2. Section 2

Description Lists

Description lists are used to display name-value pairs, like terms and their definitions.

Basic Description List

HTML
HyperText Markup Language - the standard language for creating web pages.
CSS
Cascading Style Sheets - used to style and layout web pages.
JavaScript
A programming language that enables interactive web pages.

Description List with Multiple Terms

Frontend
Client-side
The part of a website or application that users interact with directly.
Backend
Server-side
The part of a website or application that runs on the server and handles data processing.

Description List with Multiple Descriptions

Web Browser
Software application for accessing information on the World Wide Web.
Examples include Chrome, Firefox, Safari, and Edge.
Web Server
Computer software and hardware that stores, processes, and delivers web pages to clients.
Examples include Apache, Nginx, and Microsoft IIS.
