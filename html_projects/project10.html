HTML Text Formatting Elements

HTML provides several elements for formatting text. These elements add semantic meaning to your content and can also affect how the text is displayed.

Basic Text Formatting

Bold and Strong Text

The <b> element defines bold text, without any extra importance.

The <strong> element defines text with strong importance. The content inside is typically displayed in bold.

Italic and Emphasized Text

The <i> element defines text in an alternate voice or mood. The content inside is typically displayed in italic.

The <em> element defines emphasized text. The content inside is typically displayed in italic.

Underlined Text

The <u> element defines text that should be stylistically different from normal text, such as misspelled words or proper names in Chinese.

Strikethrough Text

The <s> element defines text that is no longer correct, accurate or relevant.

The <del> element defines text that has been deleted from a document. Often used to show document revisions.

Inserted Text

The <ins> element defines text that has been inserted into a document. Often used to show document revisions.

Subscript and Superscript

The <sub> element defines subscript text: H2O is the chemical formula for water.

The <sup> element defines superscript text: 23 equals 8.

Marked/Highlighted Text

The <mark> element defines text that should be marked or highlighted.

Semantic Text Elements

Abbreviations

The <abbr> element defines an abbreviation or acronym:

The WHO was founded in 1948.

Hover over the abbreviation to see the full title.

Address

The <address> element defines contact information for the author/owner of a document:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Blockquote

The <blockquote> element defines a section that is quoted from another source:

For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.

Inline Quote

The <q> element defines a short inline quotation:

WWF's goal is to: Build a future where people live in harmony with nature.

Code

The <code> element defines a piece of computer code:

The HTML <button> tag defines a clickable button.

Keyboard Input

The <kbd> element defines keyboard input:

Press Ctrl + C to copy text.

Sample Output

The <samp> element defines sample output from a computer program:

File not found.
Press F1 to continue

Variable

The <var> element defines a variable in programming or in a mathematical expression:

The area of a triangle is: 1/2 × b × h, where b is the base, and h is the height.

Definition

The <dfn> element defines a definition term:

HTML is the standard markup language for creating web pages.

Combining Text Formatting

You can combine multiple text formatting elements:

This is strongly emphasized text.

H2O is the chemical formula for water.

This is marked and deleted text.

The Evolution of Web Development

Web development has changed dramatically since the early days of the internet. What started with simple HTML pages has evolved into complex applications using frameworks like React and Angular.

As The History of Web Design notes:

The first websites were text-only with simple layouts. The introduction of CSS in 1996 revolutionized how designers approached web design, allowing for separation of content and presentation.

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>

</body>
</html>
