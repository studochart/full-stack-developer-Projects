Weather App

Create a weather application using JavaScript and APIs

Introduction

In this project, we'll build a weather application that fetches and displays weather data from a public API. The app will allow users to search for weather information by city name, view current weather conditions, and see a forecast for the next few days.

This project demonstrates how to work with external APIs, handle asynchronous JavaScript, and create a user-friendly interface for displaying data.

How It Works

The Weather App is built using the following components:

  • OpenWeatherMap API: The app uses the OpenWeatherMap API to fetch weather data for different locations.
  • Fetch API: JavaScript's Fetch API is used to make HTTP requests to the weather API.
  • Async/Await: Modern JavaScript async/await syntax is used to handle asynchronous operations.
  • Local Storage: The browser's localStorage API is used to save and retrieve the user's favorite locations.
  • DOM Manipulation: JavaScript is used to update the UI with the fetched weather data.

The app includes the following features:

  • Search for weather by city name
  • Display current weather conditions (temperature, humidity, wind speed, etc.)
  • Show a 5-day weather forecast
  • Toggle between metric and imperial units
  • Save favorite locations for quick access
  • Responsive design that works on both desktop and mobile devices
