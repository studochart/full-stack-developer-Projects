Introduction to Arrays and Objects

Arrays and objects are complex data structures in JavaScript that allow you to store and organize multiple values. They are essential for working with collections of data and representing real-world entities in your code.

In this project, we'll explore how to create, access, and manipulate arrays and objects, as well as common methods and operations for working with these data structures.

Arrays

An array is an ordered collection of values. Each value in an array is called an element, and each element has a numeric position called its index. Arrays in JavaScript are zero-indexed, which means the first element is at index 0.

Creating Arrays

// Array literal notation const fruits = ["Apple", "Banana", "Orange"]; // Using the Array constructor const numbers = new Array(1, 2, 3, 4, 5); // Empty array const emptyArray = []; // Array with mixed data types const mixedArray = [1, "Hello", true, null, { name: "John" }];

Accessing Array Elements

const fruits = ["Apple", "Banana", "Orange"]; // Accessing by index console.log(fruits[0]); // "Apple" console.log(fruits[1]); // "Banana" console.log(fruits[2]); // "Orange" // Length property console.log(fruits.length); // 3 // Accessing the last element console.log(fruits[fruits.length - 1]); // "Orange"

Modifying Arrays

const fruits = ["Apple", "Banana", "Orange"]; // Changing an element fruits[1] = "Mango"; console.log(fruits); // ["Apple", "Mango", "Orange"] // Adding elements fruits.push("Grape"); // Add to the end console.log(fruits); // ["Apple", "Mango", "Orange", "Grape"] fruits.unshift("Strawberry"); // Add to the beginning console.log(fruits); // ["Strawberry", "Apple", "Mango", "Orange", "Grape"] // Removing elements fruits.pop(); // Remove from the end console.log(fruits); // ["Strawberry", "Apple", "Mango", "Orange"] fruits.shift(); // Remove from the beginning console.log(fruits); // ["Apple", "Mango", "Orange"] // Splice - remove or replace elements fruits.splice(1, 1); // Remove 1 element at index 1 console.log(fruits); // ["Apple", "Orange"] fruits.splice(1, 0, "Kiwi", "Banana"); // Add elements at index 1 console.log(fruits); // ["Apple", "Kiwi", "Banana", "Orange"]

Array Methods

JavaScript arrays have many built-in methods that make it easier to work with them.

Iteration Methods

const numbers = [1, 2, 3, 4, 5]; // forEach - executes a function for each element numbers.forEach(function(number) { console.log(number); }); // map - creates a new array with the results of calling a function on every element const doubled = numbers.map(function(number) { return number * 2; }); console.log(doubled); // [2, 4, 6, 8, 10] // filter - creates a new array with elements that pass the test const evenNumbers = numbers.filter(function(number) { return number % 2 === 0; }); console.log(evenNumbers); // [2, 4] // reduce - reduces the array to a single value const sum = numbers.reduce(function(total, number) { return total + number; }, 0); console.log(sum); // 15 // find - returns the first element that passes the test const found = numbers.find(function(number) { return number > 3; }); console.log(found); // 4 // some - tests if at least one element passes the test const hasEven = numbers.some(function(number) { return number % 2 === 0; }); console.log(hasEven); // true // every - tests if all elements pass the test const allPositive = numbers.every(function(number) { return number > 0; }); console.log(allPositive); // true

Other Useful Array Methods

const fruits = ["Apple", "Banana", "Orange", "Mango"]; // join - joins all elements into a string console.log(fruits.join(", ")); // "Apple, Banana, Orange, Mango" // slice - returns a shallow copy of a portion of an array console.log(fruits.slice(1, 3)); // ["Banana", "Orange"] // concat - merges two or more arrays const moreFruits = ["Kiwi", "Grape"]; console.log(fruits.concat(moreFruits)); // ["Apple", "Banana", "Orange", "Mango", "Kiwi", "Grape"] // indexOf - returns the first index at which a given element can be found console.log(fruits.indexOf("Orange")); // 2 // includes - determines whether an array includes a certain element console.log(fruits.includes("Mango")); // true // sort - sorts the elements of an array console.log(fruits.sort()); // ["Apple", "Banana", "Mango", "Orange"] // reverse - reverses the order of the elements console.log(fruits.reverse()); // ["Orange", "Mango", "Banana", "Apple"]

Objects

Objects in JavaScript are collections of key-value pairs. They allow you to store related data and functionality together.

Creating Objects

// Object literal notation const person = { firstName: "John", lastName: "Doe", age: 30, email: "john.doe@example.com" }; // Using the Object constructor const car = new Object(); car.make = "Toyota"; car.model = "Camry"; car.year = 2020; // Empty object const emptyObject = {};

Accessing Object Properties

const person = { firstName: "John", lastName: "Doe", age: 30 }; // Dot notation console.log(person.firstName); // "John" console.log(person.age); // 30 // Bracket notation console.log(person["lastName"]); // "Doe" // Using variables with bracket notation const propertyName = "age"; console.log(person[propertyName]); // 30

Modifying Objects

const person = { firstName: "John", lastName: "Doe", age: 30 }; // Adding properties person.email = "john.doe@example.com"; person["phone"] = "555-1234"; // Modifying properties person.age = 31; // Deleting properties delete person.phone; console.log(person); // { // firstName: "John", // lastName: "Doe", // age: 31, // email: "john.doe@example.com" // }
