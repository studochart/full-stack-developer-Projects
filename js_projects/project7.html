JavaScript Timing Functions

Use setTimeout and setInterval to control when code executes

Introduction to Timing Functions

JavaScript provides timing functions that allow you to execute code at specified time intervals. These functions are essential for creating animations, updating content periodically, implementing countdowns, and more.

In this project, we'll explore the two main timing functions in JavaScript: setTimeout and setInterval, and see how they can be used to create various timing-based effects.

setTimeout

The setTimeout function executes a specified function once after a specified delay (in milliseconds).

// Basic syntax setTimeout(function, delay, param1, param2, ...); // Example: Execute a function after 2 seconds setTimeout(function() { console.log("This message appears after 2 seconds"); }, 2000); // With parameters setTimeout(function(name) { console.log("Hello, " + name); }, 2000, "John"); // Using a named function function greet(name) { console.log("Hello, " + name); } setTimeout(greet, 2000, "John"); // Storing the timeout ID const timeoutId = setTimeout(function() { console.log("This may never run"); }, 5000); // Canceling the timeout clearTimeout(timeoutId);

setInterval

The setInterval function repeatedly executes a specified function at specified time intervals (in milliseconds).

// Basic syntax setInterval(function, interval, param1, param2, ...); // Example: Execute a function every 1 second setInterval(function() { console.log("This message appears every second"); }, 1000); // With parameters setInterval(function(count) { console.log("Count: " + count); }, 1000, 42); // Using a named function function updateClock() { const now = new Date(); console.log(now.toLocaleTimeString()); } setInterval(updateClock, 1000); // Storing the interval ID const intervalId = setInterval(function() { console.log("Tick"); }, 1000); // Canceling the interval clearInterval(intervalId);
