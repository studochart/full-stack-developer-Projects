JavaScript Functions

Learn how to create and use functions, parameters, and return values

Introduction to Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a reusable block of code designed to perform a particular task. Functions help organize code, make it reusable, and easier to maintain.

In this project, we'll explore different ways to define and use functions in JavaScript, including function parameters, return values, and various function types.

Function Declaration

A function declaration defines a named function. Function declarations are hoisted, which means they can be called before they are defined in the code.

// Function Declaration function greet() { console.log("Hello, world!"); } // Calling the function greet(); // Outputs: "Hello, world!"

Function Parameters

Functions can take parameters, which are values you can pass to the function when you call it.

// Function with parameters function greet(name) { console.log("Hello, " + name + "!"); } // Calling the function with an argument greet("John"); // Outputs: "Hello, John!" // Function with multiple parameters function add(a, b) { return a + b; } // Calling the function with multiple arguments let sum = add(5, 3); // sum = 8

Default Parameters

// Function with default parameters function greet(name = "Guest") { console.log("Hello, " + name + "!"); } greet(); // Outputs: "Hello, Guest!" greet("John"); // Outputs: "Hello, John!"

Rest Parameters

// Function with rest parameter function sum(...numbers) { let total = 0; for (let number of numbers) { total += number; } return total; } console.log(sum(1, 2, 3, 4, 5)); // Outputs: 15

Return Values

Functions can return values using the return statement. When a function returns a value, it can be assigned to a variable or used in an expression.

// Function that returns a value function add(a, b) { return a + b; } // Using the returned value let sum = add(5, 3); // sum = 8 console.log(sum); // Outputs: 8 // Function with conditional return function getGrade(score) { if (score >= 90) { return "A"; } else if (score >= 80) { return "B"; } else if (score >= 70) { return "C"; } else if (score >= 60) { return "D"; } else { return "F"; } } let grade = getGrade(85); // grade = "B"

Return Values Example

Function Expressions

A function expression defines a function as part of a larger expression, typically a variable assignment. Function expressions are not hoisted, so they cannot be used before they are defined.

// Function Expression const greet = function() { console.log("Hello, world!"); }; // Calling the function greet(); // Outputs: "Hello, world!" // Function Expression with parameters const add = function(a, b) { return a + b; }; let sum = add(5, 3); // sum = 8

Arrow Functions

Arrow functions provide a shorter syntax for writing function expressions. They are anonymous functions with a special syntax that uses an "arrow" (=>) to separate the parameters from the function body.

// Arrow Function const greet = () => { console.log("Hello, world!"); }; // Arrow Function with parameters const add = (a, b) => { return a + b; }; // Shorter syntax for single-expression functions const add = (a, b) => a + b; // Arrow Function with a single parameter const square = x => x * x;

Arrow Functions Example

Callback Functions

A callback function is a function passed as an argument to another function, which is then invoked inside the outer function. Callbacks are commonly used in asynchronous operations, event handlers, and array methods.

// Function that takes a callback function doSomethingAsync(callback) { setTimeout(() => { const result = "Operation completed"; callback(result); }, 1000); } // Using the function with a callback doSomethingAsync(function(result) { console.log(result); // Outputs: "Operation completed" after 1 second }); // Using array methods with callbacks const numbers = [1, 2, 3, 4, 5]; // map() with a callback const doubled = numbers.map(function(num) { return num * 2; }); // Or with an arrow function const doubled = numbers.map(num => num * 2); // doubled = [2, 4, 6, 8, 10]
