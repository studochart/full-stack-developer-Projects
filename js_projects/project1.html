Introduction to JavaScript

JavaScript is a programming language that allows you to implement complex features on web pages. It's an essential technology for web development, enabling interactive web pages and is an essential part of web applications.

In this project, we'll explore the basic concepts of JavaScript, including variables, data types, and operators.

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const.

// Using var (older way, function-scoped) var name = "John"; // Using let (block-scoped, can be reassigned) let age = 25; // Using const (block-scoped, cannot be reassigned) const PI = 3.14159;

Data Types

JavaScript has several built-in data types:

  • String: Text values, enclosed in quotes
  • Number: Numeric values
  • Boolean: true or false
  • Undefined: A variable without a value
  • Null: An intentional absence of any object value
  • Object: Collections of related data
  • Symbol: Unique and immutable primitive values
  • BigInt: Integers with arbitrary precision
// String let name = "John"; // Number let age = 25; let price = 19.99; // Boolean let isActive = true; // Undefined let car; // Null let empty = null; // Object let person = { firstName: "John", lastName: "Doe", age: 30 }; // Array (type of object) let colors = ["red", "green", "blue"]; // Symbol let id = Symbol("id"); // BigInt let bigNumber = 1234567890123456789012345n;

Operators

JavaScript has various types of operators:

Arithmetic Operators

// Addition let sum = 5 + 3; // 8 // Subtraction let difference = 10 - 4; // 6 // Multiplication let product = 6 * 7; // 42 // Division let quotient = 20 / 5; // 4 // Modulus (remainder) let remainder = 10 % 3; // 1 // Increment let x = 5; x++; // x is now 6 // Decrement let y = 8; y--; // y is now 7 // Exponentiation let power = 2 ** 3; // 8 (2 raised to the power of 3)

Assignment Operators

let x = 10; x += 5; // Same as x = x + 5 x -= 3; // Same as x = x - 3 x *= 2; // Same as x = x * 2 x /= 4; // Same as x = x / 4 x %= 3; // Same as x = x % 3

Comparison Operators

// Equal to (value only) 5 == "5" // true // Equal to (value and type) 5 === "5" // false // Not equal to (value only) 5 != "6" // true // Not equal to (value and type) 5 !== "5" // true // Greater than 10 > 5 // true // Less than 5 < 10 // true // Greater than or equal to 10 >= 10 // true // Less than or equal to 5 <= 10 // true

Logical Operators

// AND (&&) true && true // true true && false // false // OR (||) true || false // true false || false // false // NOT (!) !true // false !false // true
